# JSON-TreeView Converter

The JSON-TreeView Converter is a JavaScript project that allows users to input JSON data and convert it into an interactive HTML TreeView UI. Users can move the HTML elements up, down, left, and right to update the Tree structure.

## Usage

To use the JSON-TreeView Converter, follow these steps:

1. Clone the repository onto your local machine.
2. Open the index.html file in a web browser.
3. Enter the JSON data into the text input field and click the "Convert" button.
4. The JSON data will be converted into an interactive HTML TreeView UI.

## Features
The JSON-TreeView Converter offers the following features:

- Converts JSON data into an interactive HTML TreeView UI.
- Allows users to move HTML elements up, down, left, and right to update the Tree structure.
- Supports various JSON data structures, including arrays and nested objects.
